### Reflection
- I found this tutorial while looking for tutorials to work on to continue my learning of code. I've done some HTML and CSS coding before and felt very comfortable writing those two files. However, I don't know JavaScript, and I thought this tutorial would be a good start to learning JavaScript. I found JavaScript to be a very easy language to follow as well as learn.
- I followed the tutorial and learned how to use the console on Chrome, which was the browser that I was using to see the project. Once the program that I wrote matched the end project that was shown at the end of the tutorial, I decided to add a counter above the buttons on the top so that the user can see how many rounds of the Pomodoro timer they have to do before the long break button appears between the short break and the Pomodoro button.
